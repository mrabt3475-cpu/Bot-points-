# Pointporn Wallet Bot

🤖 بوت محفظة إلكترونية مشفرة

## الميزات
- 💰 محفظة نقاط لكل مستخدم
- 🛒 شراء نقاط عبر باينس
- 📤 إرسال نقاط لصديق
- 📥 استقبال نقاط
- 🔗 نظام إحالة (20% عمولة)

## الأوامر
- /start - بدء البوت
- /balance - عرض الرصيد
- /buy - شراء نقاط
- /send - إرسال نقاط
- /referral - رابط الإحالة
- /help - مساعدة

## التشغيل
```bash
pip install -r requirements.txt
python bot.py
```
